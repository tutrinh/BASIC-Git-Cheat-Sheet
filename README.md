# Git Cheat Sheet

## Get Current Remote Repo
```text
$ git remove -v
```
## Branch
### Show Branches
```text
$ git branch
```
### New Branch Based on the Current HEAD
```text
$ git branch <new-branch>
```

### New Branch Based on Existing One
```text
$ git branch <new-branch> <base-branch>
```
